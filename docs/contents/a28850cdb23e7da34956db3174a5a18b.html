CodeList

UMLClass
Plot-X - Plot Data Exchange Standard :: Plot-X Core :: CodeList

Description

A codelist contains concepts widely adopted by their domain-specific community with URIs as identifiers. Codelists generally follow Linked Data best practices and uses Simple Knowledge Organization System (SKOS) and/or the Web Ontology Language (OWL) to describe the concepts.
